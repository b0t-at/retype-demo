# Welcome to Retype Demo

![Retype](https://retype.com/static/retype-icon.svg){ width="100" }

This is a comprehensive demonstration of [Retype](https://retype.com), an ultra-high-performance static site generator that builds a website based on simple Markdown (.md) text files.

## What is Retype?

Retype is a modern static site generator that transforms your Markdown files into beautiful, fast, and functional websites. Focus on your writing while Retype builds the rest.

### Key Features

- :rocket: **Ultra-fast** - Lightning-fast performance
- :memo: **Markdown-based** - Simple and intuitive content creation
- :art: **Beautiful themes** - Professional and customizable designs
- :zap: **Live reload** - See changes instantly during development
- :mag: **Built-in search** - Powerful full-text search functionality
- :sparkles: **Rich components** - Alerts, tabs, panels, and more

## Quick Start

Get started with Retype in just a few steps:

1. Install Retype
   ```bash
   npm install retypeapp --global
   ```

2. Initialize your project
   ```bash
   retype init
   ```

3. Start the local server
   ```bash
   retype watch
   ```

4. Build for production
   ```bash
   retype build
   ```
